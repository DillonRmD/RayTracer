# RayTracer
This repository is a implementation of the very famous [Ray Tracing in One Weekend Book](https://raytracing.github.io/books/RayTracingInOneWeekend.html) by Peter Shirley.

This project is meant to be a learning experience in order to get my feet wet on more graphics related programming.
This project also doubles as a way to put my [math library](https://github.com/DillonRmD/dtb/blob/master/dtb_math.h) to the test and update it.

A lot of the initial code is derived straight from the book; however, in later iterations it is planned to make a more personal take on the topic of ray tracing.
